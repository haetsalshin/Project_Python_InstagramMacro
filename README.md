# :camera:Project_Python_InstagramMacro:camera:

Python 기반의 Selenium(+feat : Chrome Driver) 사용하여 인스타그램에 로그인하고, 원하는 해시태그로 피드를 검색한 후 피드별로 좋아요를 클릭하고 댓글 등록을 반복하는 매크로 프로그램.

## :heavy_check_mark:Developer Environment
